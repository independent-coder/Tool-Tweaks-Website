Welcome to the official homepage of Tool Tweaks. Please explore the links on the left to learn more about Tool Tweaks. DISCLAIMER: Every Tool Tweaks app is available on GitHub under the Independent-Coder account.

September 4, 2023: Encoding Tweaks is a Python CLI Application that was released on September 4, 2023, and offers a lot of possibilities. It can handle text encoding in multiple formats, text decoding from multiple formats, color display for encoded text, and supports encoding and decoding text in different UTF formats.

August 31, 2023: We are excited to announce the release of our first Tool Tweaks application, Hashing Tweaks. This Python CLI application has been created on GitHub. Hashing Tweaks serves multiple purposes, including finding common passwords using hashes, generating hashes from text or passwords, translating between hash algorithms, creating hashes from files, and comparing two hashes. It also supports (MD5, SHA-256, SHA3-256)
